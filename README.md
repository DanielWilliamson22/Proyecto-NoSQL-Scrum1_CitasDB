# Proyecto-NoSQL-Scrum1_CitasDB

Desarrollar una aplicación web para la gestión de citas médicas (o de servicios), permitiendo a los usuarios seleccionar una sede, fecha y hora disponible, almacenando la información en una base de datos NoSQL (MongoDB).
